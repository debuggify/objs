h2. Description

A minimalistic library intended to help in using namespaces and class inheritance in JavaScript.

Objs enables:
* Class declaration using namespaces
* Access to class constructors using their namespaces
* Simplified inheritance syntax which follow ExtJS, Prototype, Mootoolsâ€¦ conventions
* Shortcuts to superclass and superclass methods 
* A complete library to help in using OOP for only 740 chars
